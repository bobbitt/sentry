
                              Sentry V6.5
                          PC Security System

Sentry is an extremely flexible security program allows complete
customization to fit your needs. Protect system, O/S, and/or individual
files and support up to 100 users with normal and "SuperUser" access.
Detailed log files, a message system, auditing, expiry dates, and
non-recoverable deletion help with security and system administration.
Network/DESQview compatible.
